## 🚀 Sobre o Projeto

Este é o meu portfólio pessoal de desenvolvimento Front-end, desenvolvido para apresentar o meu perfil profissional, as minhas competências e os projetos mais relevantes em que tenho trabalhado. O design foi planeado para oferecer uma experiência limpa, moderna e totalmente responsiva (adaptada a computadores e telemóveis).

A aplicação encontra-se publicada e pode ser visualizada em tempo real através do link:
👉 **[portifolio-nu-azure-38.vercel.app](https://portifolio-nu-azure-38.vercel.app)**

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento Web:

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização moderna, utilização de variáveis, Flexbox e Grid Layout para responsividade.
- **JavaScript (ES6):** Manipulação do DOM, interatividade e integração de funcionalidades.
- **Git & GitHub:** Controlo de versões e gestão do código fonte.
- **Vercel:** Plataforma utilizada para o deployment e alojamento rápido da aplicação.
